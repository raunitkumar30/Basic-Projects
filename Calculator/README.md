# Modern Web Calculator

A simple, interactive, and beautiful web-based calculator styled with a premium dark theme, glassmorphism-inspired components, elegant typography, and responsive transitions.

## Features

- **Premium Design**: Modern dark theme utilizing standard Google Fonts (`Plus Jakarta Sans` and `JetBrains Mono`).
- **Tactile Feedback**: Interactive buttons with custom hover, active click scale, and shadows.
- **Complete Arithmetic**: Supports addition, subtraction, multiplication, division, percentage, and backspace operations.
- **Responsive Layout**: Adapts cleanly to different screens.

## Project Structure

- `index.html` - The markup and structure of the calculator layout.
- `style.css` - Custom premium dark-themed styles.
- `script.js` - Calculator logic and event handlers.

## How to Run

Simply open `index.html` in any web browser, or serve it using a local server (e.g. Live Server).
